# myFlix-client
This is thr client-side interface and application for a movie application based off a server-side REST API database, [myFlix-app](https://github.com/Coldest-Out/myFlix-app). Users have the ability to register, log-in, add/remove their favorite movies from the given selection, and can delete their accounts when they'd like too.

## Built with:
- React
- React-Bootstrap
- Redux
- Javascript
- SCSS

## Depedencies
- Axios
- Proptypes
- React
- React-Bootstrap
- React-DOM
- React-router-dom
- Redux
